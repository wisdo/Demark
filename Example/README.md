## Running the Example

### Quick Start

```bash
# Use the convenience script from project root
./run-example.sh

# Or run manually
cd Example
swift run DemarkExample
```

### Development in Xcode

```bash
# Generate and open Xcode project  
./generate-xcodeproj.sh --open

# Or open Package.swift directly
open Package.swift
```

## Requirements
